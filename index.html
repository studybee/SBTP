Studybee Template

이 페이지는 studybee "서비스 만들기" 클래스에서 사용할 공통 위젯을 올려놓는 페이지 입니다.

  • 1. 공통 Class는 "sb-" 를 prefix로 합니다.
  • 2. prefix 뒤의 클래스 이름은 /css/클래스명/main.css 로 생성 후 작성합니다.
  • 3. 공통 class가 아닌 해당 페이지를 위한 css는 /assets/css/main.css 에 작성합니다.

Github : https://github.com/studybee/SBTP
